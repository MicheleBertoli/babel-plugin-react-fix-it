# Babel Plugin React Fix It
Babel plugin for [react-fix-it](https://github.com/MicheleBertoli/react-fix-it).

## Installation

```
yarn add --dev react-fix-it babel-plugin-react-fix-it
```

## Usage

```json
{
  "env": {
    "development": {
      "plugins": ["react-fix-it"]
    }
  }
}
```

## Test

```
yarn test
```
